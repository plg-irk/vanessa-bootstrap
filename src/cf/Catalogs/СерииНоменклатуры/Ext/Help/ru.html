Справочник "Серии номенклатуры"

Справочник предназначен для хранения информации о сериях номенклатуры.

Использование справочника зависит от параметра учета системы "Использовать серии номенклатуры". Значение параметра  устанавливается в разделе "Еще больше возможностей" - "Закупки".

  • Переключатель "Использовать серии" – Включает использование серий для экземпляра номенклатуры или категории. 

    Если значение переключателя "Общие для категории", тогда все настройки серий для подчиненной номенклатуры будут браться из категории. Включая непосредственно сами серии.

    Поле "Политика учета серий" - Обязательное для заполнение поле. Необходимо для выбора правил заполнения серий номенклатуры в документах.
  • Флаг "Серийные номера" - Определяет экземпляр товара, имеющий уникальный серийный номер.
  • Флаг "Сроки годности" - Включает указание сроков годности в сериях номенклатуры.
  • Флаг "Дата производства" - Включает указание даты производства в сериях номенклатуры.
  • Поле "Точность указания даты" - Определяет формат даты при указании срока годности и даты производства.
  • Поле "Срок годности" - Определяет длительность срока годности в заданных единицах времени.
  • Группа "Индивидуальные настройки по складам" - Позволяет вводить индивидуальные настройки политик учета серий в разрезе склада и организации.

Флаг "Продан" в строке списка устанавливается автоматически при реализации серий с установленным флагом "Серийные номера".

В таблице индивидуальных настроек по складам указывается уточнение политик учета по складам в разрезе организаций. Если уточнение на задано, тогда основная политика учета из настроек будет действовать для всех складов и организаций.
