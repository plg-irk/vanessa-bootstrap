Документ "Сдельный наряд"

Документ предназначен для планирования и указания факта выполнения операций бригадой или сотрудником. Увеличение себестоимости изделий/работ сдельным нарядом возможно только после включения раздела "Зарплата" в настройках программы.

 

В документе необходимо заполнить следующие реквизиты:

  • Организация, от лица которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Подразделение, в котором оформляется документ. Выбирается из справочника "Структурные единицы". Есть возможность переключения режима отображения, при котором подразделение будет задаваться для каждой операции отдельно.
  • Исполнитель, по которому оформляется документ. Выбирается из справочника "Сотрудники" или "Бригады". Если исполнителем выбрана бригада, то для начисления можно указать состав бригады и КТУ на закладке "Состав бригады". Есть возможность переключения режима отображения, при котором исполнитель будет задаваться для каждой операции отдельно. В таком режиме состав бригады и КТУ также индивидуальны для каждой операции.
  • Заказ на производство, по которому выполнена операция. Выбирается из списка документов "Заказы на производство".
  • Валюта документа - валюта, в которой указывается стоимость работ. Выбирается из справочника "Валюты".
  • Закрыт, Дата закрытия - при установленном флаге Закрыт документ начисляет зарплату сотрудникам на Дату закрытия.
  • Комментарий. Указывается любая дополнительная информация к документу.
  • Операции. В табличной части указывается запланированная и выполненная операция с возможностью детализации до заказа покупателя и продукции. Реквизиты табличной части:
    • Дата. Дата выполнения операции, на нее будет начислена сумма зарплаты.
    • Этап. При использовании поэтапного производства указывается этап производства, к которому относится выполняемая операция. При отключенной опции колонка не отображается.
    • Заказ покупателя, по которому выполнена операция. Выбирается из списка документов "Заказы покупателей".
    • Заказ на производство см. такое же поле в шапке документа. Отображается при переключении режима расположения заказа на производство через пункт "Шапка / табличная часть" подменю "Еще" или контекстного меню таблицы операций.

    • Номенклатура, для изготовления которой выполнена операция. Выбирается из справочника "Номенклатура".

    • Характеристика номенклатуры, для изготовления которой выполнена операция. Выбирается из справочника "Характеристики номенклатуры".
    • Партия номенклатуры, для изготовления которой выполнена операция. Выбирается из справочника "Партии номенклатуры".
    • Единица измерения операции. Выбирается из справочника "Единицы измерения" или "Классификатор единиц измерения".
    • Операция. Выбирается из справочника "Номенклатура".
    • Количество (план) - количество операций, запланированных к выполнению.
    • Количество (факт) - фактическое количество выполненных операций.
    • Норма - норма времени для изготовления единицы продукции.
    • Расценка - стоимость выполнения операции для изготовления единицы продукции.
    • Нормочасы - количество нормочасов.
    • Стоимость выполненных операций.
    • Исполнитель см. такое же поле в шапке документа. Отображается при переключении режима расположения исполнителя через пункт "Шапка / табличная часть" подменю "Еще" или контекстного меню таблицы операций.
    • Подразделение см. такое же поле в шапке документа. Отображается при переключении режима расположения подразделения через пункт "Шапка / табличная часть" подменю "Еще" или контекстного меню таблицы операций.
    • Завершающее подразделение нужно указывать при использовании поэтапного производства с возможностью выполнять этапы разными подразделениями. Содержит структурную единицу, на которой будет храниться остаток незавершенного производства.
       
  • Состав бригады. В табличной части указывается фактический состав бригады, выполняющей операции. В командной панели предусмотрен механизм автозаполнения списком сотрудников по умолчанию. Реквизиты табличной части:
    • Сотрудник, входящий в состав бригады. Выбирается из справочника "Сотрудники".
    • КТУ - коэффициент трудового участия, пропорционально которому делится стоимость работ. Указывается доля, а не процент.
В командной панели на закладке "Операции" расположена кнопка  ("Остатки план/факт"). Данная кнопка активирует режит отображения, показывающий количество операций запланированные документом основанием, а также запланирование и выполненные опеарации другими сдельными нарядами.
  • План по заказу / всего - Количество операций запланированных документом "Заказ покупателя", "Заказ на производство" или "Производство" / Общее количество операций запланированных к выполнению документами "Сдельный наряд".
  • Всего факт - Общее количество выполненных операций документами "Сдельный наряд".
Документ "Сдельный наряд" может быть введен на основании документов:
  • "Заказ покупателя";
  • "Заказ на производство";
  • "Производство".
